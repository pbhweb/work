# دليل نشر المشروع على Vercel

## المتطلبات
- حساب Vercel (مجاني)
- قاعدة بيانات Supabase (متصلة بالفعل)

## خطوات النشر من v0

### 1. انشر المشروع
- اضغط على زر **"Publish"** في أعلى يمين الشاشة في v0
- سيتم نشر المشروع تلقائياً على Vercel

### 2. ربط الدومين المخصص (workshub.space)

بعد النشر، اتبع هذه الخطوات:

#### في لوحة تحكم Vercel:
1. افتح مشروعك في Vercel
2. اذهب إلى **Settings** → **Domains**
3. اضغط **Add Domain**
4. أدخل: `workshub.space`
5. ستظهر لك تعليمات DNS

#### في لوحة تحكم الدومين:
أضف السجلات التالية في DNS الدومين:

**للدومين الرئيسي:**
```
Type: A
Name: @
Value: 76.76.21.21
```

**للنطاق الفرعي www:**
```
Type: CNAME
Name: www
Value: cname.vercel-dns.com
```

### 3. تشغيل سكريبتات قاعدة البيانات

بعد النشر، قم بتشغيل السكريبتات بالترتيب:

1. في v0، اذهب للمجلد `scripts`
2. شغل السكريبتات بالترتيب:
   - `001_create_database_schema.sql`
   - `002_seed_sample_data.sql`
   - `003_create_helper_functions.sql`

### 4. متغيرات البيئة الإضافية (اختياري)

إذا احتجت متغيرات إضافية، أضفها في:
**Vercel Dashboard** → **Settings** → **Environment Variables**

المتغيرات المطلوبة موجودة بالفعل من تكامل Supabase.

### 5. التحقق من النشر

بعد النشر، تحقق من:
- الصفحة الرئيسية: `https://your-project.vercel.app`
- API الصحية: `https://your-project.vercel.app/api/health`

## الميزات المتاحة

✅ نظام التسجيل والدخول (3 أنواع حسابات)
✅ نشر المشاريع لأصحاب الأعمال (حد أدنى 300 دولار)
✅ تقديم العروض للمستقلين
✅ نظام الأفلييت بعمولة 10%
✅ لوحة تحكم المعاملات المالية
✅ حماية من مشاركة معلومات التواصل
✅ نظام RLS كامل للأمان

## الدعم

إذا واجهت أي مشكلة:
1. تحقق من Logs في Vercel Dashboard
2. تأكد من تشغيل جميع السكريبتات
3. تحقق من اتصال Supabase
